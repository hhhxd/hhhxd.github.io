type=”checkbox”

<input type="checkbox">勾选框的一些样式修改无效，比如background-color，写上去没有效果。

<label>标签修改<input type="checkbox">样式

在<input>后添加<label>标签,通过<input>的id和<label>的for保持一致，实现绑定。

css: 给label标签添加选中前样式，选中后（checked）的label添加伪元素before

type=”radio”

<input type="radio"> 单选框，在点击单选框时可选中，但是无法通过点击文字来选中。

例如：

<label>标签，通过点击文字来选中单选框

`<input>`标签后面加对应`<label>`标签

或 用`<label>`标签包裹对应`<input>`标签

type=”text”

<input type="radio"> 输入框，有placeholder属性可以设置预期输入值。

预期输入值的样式修改，不同浏览器兼容：

浏览器保存了密码

如果浏览器保存了账号密码，在input获取到焦点时后，鼠标移至已保存项上~直到该项被填充至input框，在这期间预期输入值的样式会被浏览器自带的丑黄色代替掉。

解决方式1： 简单粗暴地给input一个autocomplete="off"属性，把选择历史值的功能关闭。

解决方式2： :-webkit-autofill属性，给input设置一个超大的内阴影（至少比input大）
